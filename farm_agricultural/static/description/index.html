Sistema de gestión agrícola que extiende el módulo MRP de Odoo para campañas de cultivos. Funcionalidades principales: - Planificación de campañas agrícolas - Gestión de planes de labores por cultivo - Seguimiento de aplicaciones de insumos - Control de costos y análisis de rentabilidad - Integración con gestión de campos y lotes - Reportes especializados para el sector agropecuario El módulo permite gestionar el ciclo completo desde la planificación hasta la cosecha, con seguimiento detallado de insumos, equipos y condiciones de aplicación. Desarrollado específicamente para el sector agropecuario argentino con estándares internacionales.
